**Airflow Improved** 是一个基于 [airflow-csgo-cheat by mmirraacclee](https://github.com/mmirraacclee/airflow-csgo-cheat) 的分支项目。此版本旨在引入新的功能和改进，以增强其核心能力。

## ✨ 主要特性与改进

此分支在原项目的基础上添加了以下关键特性：

*   🚀 **Lua 脚本系统 (开发中)**
    *   引入了一个 Lua 脚本系统，旨在为用户提供更高级别的自定义和扩展能力。
    *   *状态：目前尚未完成，正在积极开发中。*

*   🎯 **基于多边形的 `get_points` 函数**
    *   `get_points` 函数（用于获取目标点位）已重构，采用多边形计算方法。
    *   这可能带来更精确的多点扫描和边缘点位覆盖。

*   📈 **全新的 Hitchance (命中几率) 计算方式**
    *   实现了一种新的命中几率计算逻辑。
    *   旨在提供更准确和可靠的命中预测。
